这是我们哈尔滨工业大学2025级大一年度项目航天26号项目库，要实现AirSim环境以及实际环境下的无人机集群视觉感知以及集群控制。

## 本项目：包安装与运行说明

- **安装（可编辑安装，便于开发并在其它电脑上复现）**：在项目根目录下运行：

```powershell
python -m pip install -e .
```

- **运行测试脚本（推荐）**：在项目根目录运行模块方式，以确保项目根被加入搜索路径：

```powershell
python -m src.drone_test
```

- **说明**：我们已将 `Drone_Interface` 目录设为 Python 包（含 `__init__.py`），因此其他组员只需克隆仓库，运行上述 `pip install -e .` 即可在本机上导入 `Drone_Interface` 包。

如果不希望安装，也可以在项目根用模块方式运行脚本（如上），避免直接在 `src` 目录下运行脚本导致导入失败。
