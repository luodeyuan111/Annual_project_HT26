## 更新摘要

本次更新主要解决视觉处理质量问题，添加可视化功能，并为中期答辩做准备。

## 详细更新内容

### 1. Bug修复

#### 1.1 光流置信度计算
- **文件**: `src/neural_processing/neural_perception.py`
- **问题**: 使用光流幅度平均值判断每个像素的置信度，逻辑错误
- **修复**: 计算每个像素的光流幅度，统计在有效范围内的比例

### 2. 功能改进

#### 2.1 障碍物检测优化
- **文件**: `src/Visual_process/obstacle_detect.py`
- 实现文献中的水平切片方法（取每列最小深度）
- 改进角度映射，使用 `np.arctan2` 准确计算角度
- 添加深度裁剪，过滤不合理的深度值（<0.1m或>max_distance）

#### 2.2 位姿估计优化
- **文件**: `src/Visual_process/pose_estimator.py`
- RANSAC阈值: 1.0 → 0.5（提高鲁棒性）
- 最小点数要求: 30 → 15
- 添加最小内点比例检查（30%）
- 修复速度平滑计算的bug

#### 2.3 日志输出优化
- **文件**: `src/main.py`
- 分类清晰展示：神经网络、机器人运动、障碍物检测
- 添加分隔线和结构化输出
- 使用英文标注避免OpenCV乱码

### 3. 新增功能

#### 3.1 可视化工具类
- **文件**: `src/utils/visualization.py` (新增)
- 深度图可视化（彩色编码，带裁剪）
- 障碍物雷达图（雷达样式显示360°分布）
- 光流可视化
- 组合显示（水平拼接）

#### 3.2 可视化按键开关
- **文件**: `src/main.py`, `src/utils/keyboard_control.py`
- 按 `v` 键切换可视化显示
- 可视化开启时显示: RGB原图 + 深度图 + 障碍物雷达图

### 4. 域差异保护

#### 4.1 深度估计保护
- **文件**: `src/utils/visualization.py`
- 添加 `max_depth_clip=20.0` 参数
- 裁剪超过20米的深度值（天空等）

#### 4.2 光流估计保护
- **文件**: `src/neural_processing/neural_perception.py`, `src/neural_processing/flow_processor.py`
- 添加 `max_flow_magnitude=50.0` 裁剪
- 过滤不合理的极大光流值

**原因**: RAFT和MonoDepth2用真实场景训练，对AirSim合成图像存在域差异（Domain Gap）

## 已知问题

### 神经网络域差异
- **问题**: 神经网络模型对AirSim合成图像深度/光流估计不准确
- **影响区域**: 天空、地面、重复纹理区域
- **根本原因**: 训练数据为真实场景，与合成图像存在域差异
- **缓解措施**: 添加裁剪和过滤（已实施）
- **根本解决**: 需要重新训练或使用适配权重（超出当前项目范围）
