# 图片显示修复总结

## 问题分析

在之前的代码中，`process_two_frames` 函数没有返回值，导致主循环中的 `display_frame` 变量总是为 `None`，图像无法显示。

## 修复内容

### 1. 修改 `process_two_frames` 函数返回值

**位置**: `src/main.py` - `process_two_frames()` 函数

#### 修改前：
```python
# 处理两帧
if frame_t is not None and frame_t_plus_1 is not None:
    display_frame = frame2  # ❌ 错误：设置局部变量，不返回
    logger.info(f"[第2帧] 捕获成功 - 分辨率: {frame2.shape}, 时间戳: {timestamp2}")
    
    # ... 处理逻辑 ...
    
    # ❌ 没有返回语句
```

#### 修改后：
```python
# 处理两帧
if frame_t is not None and frame_t_plus_1 is not None:
    logger.info(f"[第2帧] 捕获成功 - 分辨率: {frame2.shape}, 时间戳: {timestamp2}")
    
    # ... 处理逻辑 ...
    
    # ✅ 返回第2帧用于显示
    return frame2
```

### 2. 添加所有错误路径的返回值

为了确保函数在所有情况下都有明确的返回值，我们在所有错误路径上都添加了 `return None`：

```python
# 成功处理：返回 frame2
if frame_t is not None and frame_t_plus_1 is not None:
    # ... 处理逻辑 ...
    return frame2  # ✅ 返回第2帧

# 异常捕获：返回 None
except Exception as e:
    logger.error(f"⚠ 视觉处理失败 - 错误: {e}", exc_info=True)
    return None  # ✅ 返回 None

# 其他错误路径：都返回 None
else:
    logger.warning("⚠ 帧数据不足，无法处理")
    return None  # ✅ 返回 None
```

### 3. 主循环接收返回值

主循环已经正确接收返回值：

```python
# 主循环中
if need_process:
    need_process = False
    display_frame = process_two_frames(  # ✅ 接收返回值
        extractor, 
        frame_buffer, 
        camera_name, 
        neural_perception, 
        visual_perception, 
        frame_counter,
        logger,
        client
    )
    frame_counter += 1

# 显示图像（如果已捕获）
if display_frame is not None:  # ✅ 检查返回值
    try:
        # AirSim 返回的是 RGB，OpenCV 使用 BGR，用于正确显示颜色
        if isinstance(display_frame, np.ndarray) and display_frame.ndim == 3 and display_frame.shape[2] == 3:
            show_frame = cv2.cvtColor(display_frame, cv2.COLOR_RGB2BGR)
        else:
            show_frame = display_frame
        cv2.imshow("Drone View", show_frame)  # ✅ 显示图像
    except Exception as e:
        print(f"显示图像时出错: {e}")
```

### 返回值设计
- **成功情况**: 返回 `frame2` (numpy数组，RGB格式)
- **失败情况**: 返回 `None`
- **类型**: `Optional[np.ndarray]`

### 颜色转换
```python
# AirSim 返回的是 RGB，OpenCV 使用 BGR
if isinstance(display_frame, np.ndarray) and display_frame.ndim == 3 and display_frame.shape[2] == 3:
    show_frame = cv2.cvtColor(display_frame, cv2.COLOR_RGB2BGR)

    show_frame = display_frame
```

### 错误处理
所有可能的错误路径都：
1. 记录详细的错误日志
2. 返回 `None` 让主循环继续运行
3. 不影响后续帧的处理
